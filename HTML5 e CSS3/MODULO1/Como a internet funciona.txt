Representação dos dados:

8 bit tem a equivalencia de 1024 bytes, pois o computador trabalhar na multiplicação por 2 elevado a 10.

MB - Megabyte = armazenamento
Mb - Megabits = transmissão

Conexão a internet

Cliente -> Modem(agência telefonica)  -> Internet -> DNS(Altera o dominio pelo IP) -> Acesso ao servidor

iplocation -> Descobrir seu IP e de alguns sites

Dominio e hospedagem

Dominio: nome pra identificar o site.
-> nome unico não existe mais de um com o mesmo nome.
-> Pago (anual)
-> TLD a escolher

URL - buscar significado

Exemplo:

www.github.com/lfsilva92

www -> sub-dominio
github.com -> Dominio
.com -> TLD

**GTLD: Dominios genericos
**CCTLD: Dominios da nacionalidade da origem do site e/ou sistema

/lfsilva92 -> caminho
/ -> home principal



Hospedagem: Guardar os arquivos do site criado.
-> Pago (mensalmente)
-> Seleciona o pacote pra uso do site e/ou sistema
